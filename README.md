Run backend:

\`\`\`bash
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
\`\`\`

Run frontend:

\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
